Архитектура итогового проекта
    Цель: реализовать работу стропольщика
    Задачи: по входному изображению подаваемых сигналов стропольщика определить дальнейшее действие

    Требования: вэб приложение, определяющие СИЗ по фото
    Входные данные: изображению подаваемых сигналов стропольщика
    Выходные данные: картинка дальнейших действий


Задание магистрской:
"Реализация стропальщика для компьютерного тренажерного комплекса".
Суть работы: Модуль компьютерного тренажера портального крана для управления поведением персонала порта (стропальщиков).
Стропальщик должен подавать руками сигналы для оператора, которые зависят от того, в какой стадии перегрузки сейчас находится груз,
 а также должен перемещаться по сцене, выполнять строповку грузов.

Примерный стек технлогий: flask, opencv, нейронка для распознавания жестов
